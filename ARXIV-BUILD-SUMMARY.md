### Upload Steps (Per Paper)
1. **Upload Zip File**
   - Navigate to https://arxiv.org/submit
   - Upload the appropriate `arxiv-*-source.zip` file
   - Select "LaTeX" as the file type

2. **Verify arXiv Preview**
   - Wait for arXiv's automated compilation
   - Check the preview PDF for:
     - Correct title and author
     - Proper abstract formatting
     - All sections rendered correctly
     - All figures displayed

3. **Fix Any Issues**
   - If fonts are missing: arXiv will auto-substitute
   - If packages are missing: Only standard packages were used
   - If compilation fails: Check the arXiv log for specific errors

4. **Final Verification**
   - Confirm PDF is text-searchable (standard for pdflatex)
   - Verify all figures appear correctly
   - Check that references are properly formatted
   - Ensure no "??" appears in citations

5. **Submit Metadata**
   - Enter the arXiv title (from metadata document)
   - Enter abstract (150-220 words)
   - Select primary category (cs.DC, cs.SE, etc.)
   - Add secondary categories if applicable
   - Enter keywords
