\begin{abstract}
This paper introduces the A5 Sovereign Migration framework for transitioning legacy applications to multi-cloud sovereign architectures. Through eBPF-based discovery, strangler-fig patterns, and shadow validation, we achieve 70\% reduction in migration lead time with zero data loss.
\end{abstract}

\keywords{cloud migration, strangler pattern, eBPF, modernization}

\section{Introduction}
Legacy modernization faces Data Gravity, undocumented dependencies, and regulatory constraints. A5 provides a five-phase methodology for risk-controlled migration.

\section{The Strangler Fig Architecture}
The Strangler Facade intercepts traffic, enabling incremental extraction of domains with Anti-Corruption Layers preventing legacy debt leakage.

\section{Production Results}
Global logistics: 14 months (vs 2.5 years), 30\% cost reduction. Financial services: 450 TB migrated, 99.999\% validation match.
